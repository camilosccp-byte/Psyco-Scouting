# Psyco-Scouting: Hybrid Scouting Algorithm in R

## 📌 Project Overview
This project introduces an innovative, hybrid approach to football recruitment. It bridges the gap between pure eventing data and human behavior by integrating tactical performance metrics with psychosocial variables. The goal is to provide recruitment departments with a multi-dimensional tool to evaluate not just how a player performs, but how they might adapt and mentally handle professional environments.

## 📊 Methodology & Technical Stack
- **Language:** R
- **Data Modeling:** Utilized Poisson and Bernoulli distributions to process and normalize eventing/performance data.
- **Advanced Analytics:** Implemented Decision Trees (Árboles de Decisión) to map and blend psychosocial profiles with on-pitch metrics.
- **Key Focus:** Predictive talent identification and profiling.

## 🚀 Key Insights for Football Clubs
1. **Holistic Recruitment:** Moves away from one-dimensional stats to understand the player behind the numbers.
2. **Risk Mitigation:** Helps sporting directors evaluate mental readiness alongside tactical compliance.
3. **Data-Driven Scouting:** Filters massive datasets into clear, actionable scouting trees.
